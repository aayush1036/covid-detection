Introduction

This project aims for automating the process of detecting covid from medical x-rays and building a web app where we can upload the x-ray and detect covid as reports take some time to generate, we can give the patient some idea instantly so they can get treatment accordingly, main aim is to speed up the process of covid detection.

Know Your Data

The dataset used in this project is sourced from Kaggle which is the Covid chest x-ray dataset Since the data was not in a desirable format, we have converted it into a desriable format and created training, testing and validation sets, to do this, we have applied two successive train test splits, which are:
  1. Train test split on the entire data to create training and testing sets, with 20% data in the testing set and 80% data in the training set
  2. Train test split on the training set to create training and validation sets, with 10% data in the validation set and 90% data in the training set
